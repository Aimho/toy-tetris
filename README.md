## 사용할 기술
- Classes
- Arrow functions
- Spread operator

- Proxy
- Default parameters
- Array.from()

- board.js: 보드 로직
- piece.js: 테트리스 조각 로직
- main.js: 게임 초기화, 종료 로직
- index.html
- styles.css
- README.md
